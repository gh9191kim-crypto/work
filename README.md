# RGB OLED 전도도(Conductivity) 거동 시뮬레이터

RGB OLED의 각 layer(HIL·HTL·EML·ETL·EIL)의 **전도도(conductivity / 저항)** 를 바꿨을 때
전자·홀의 이동과 **Lateral Leakage**가 저계조 화질지표인 **FFR**과 **Gray Crushing**에
어떤 영향을 주는지 **시각적으로** 이해하기 위한 교육용 웹 시뮬레이터입니다.

> 👉 **바로 실행:** `index.html` 파일을 브라우저로 열면 됩니다. (설치 불필요, 인터넷 불필요)
> 공유용으로 GitHub Pages에 올리면 링크 하나로 누구나 접속할 수 있습니다.

## 무엇을 보여주나

### 단면 거동 시각화
- R / G / B 서브픽셀 단면(`Anode → HIL → HTL → EML → ETL → EIL → Cathode`).
- **홀(빨강)** 은 Anode에서, **전자(파랑)** 는 Cathode에서 주입되어 EML에서 **재결합(노란 발광)** 합니다.
- 입자의 양·속도는 전류(전도도)에 비례합니다.
- **공통층(HTL·ETL)** 의 전도도가 높으면 캐리어가 옆 서브픽셀로 새는 **Lateral Leakage**(점선 ↔)가 보입니다.

### ① FFR 측정 (First Frame Response)
- 전체 **BLACK → WHITE** 전환 시 한 프레임 안에 White 100%에 도달하는지 측정.
- 충전 시정수 `τ = C·ΔV / I` 기반. 전도도가 낮거나 저계조여서 전류가 작으면
  τ가 커져 첫 프레임에 100%에 못 미칩니다(FFR 불량).

### ② Gray Crushing 측정
- R/G/B 각 색을 단독 점등했을 때의 휘도를 **누설이 없는 이상값과 비교**.
- Lateral leakage가 크면 목표 색 EML로 갈 전류가 이웃으로 새어
  **휘도비가 100% 아래로 떨어지고**(저계조 색틀어짐) 이웃 화소가 약하게 켜집니다.

## 두 가지 물리 모델

| 모델 | 설명 | 용도 |
|------|------|------|
| **개념 모델** | 0~100% 정규화된 직관 모델(닫힌 형식). | 경향을 빠르고 부드럽게 이해 |
| **정량 모델** | 1차원 **저항–다이오드 네트워크**(단위 S/cm, 3 서브픽셀). 노드 전압을 Newton 반복으로 해석. | 단위·비대칭 누설 등 정량적 경향 |

> ⚠️ 두 모델 모두 **교육용 근사**입니다. 절대 수치가 아니라 **경향(trend)** 해석에 사용하세요.
> EML 전류는 발산 방지를 위해 SCLC형 멱함수(`I ∝ (V−Vk)²`)로 근사했습니다.

## 조작 방법
1. 상단 탭에서 **FFR** / **Gray Crushing** 선택.
2. **물리 모델**(개념/정량) 선택.
3. 각 **Layer 전도도 σ** 슬라이더 조절 → 시각화·수치 실시간 갱신.
4. **계조(Gray Level)** 를 낮춰 저계조에서의 열화를 관찰.
5. FFR 탭: **⚡ BLACK→WHITE 전환** 버튼으로 첫 프레임 거동 확인.
   Gray 탭: R/G/B/White 시나리오 선택.

## 관찰 포인트(실험 예시)
- **HTL·ETL(공통층) σ ↑** → Lateral leakage ↑ → Gray Crushing 휘도비 ↓, 이웃 화소 점등.
- **EML·수송층 σ ↓** + **저계조** → 전류 ↓ → τ ↑ → FFR ↓ (한 프레임에 흰색 못 올라옴).
- **주사율 ↑(120/240Hz)** → 프레임시간 ↓ → FFR 더 어려워짐.
